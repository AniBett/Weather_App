## About The Project
Project to practice API calls, along with Asychronous JavaScript
### Features
* Get current day and weekly forecast.
* Search cities from around the globe.
* Switch between imperial and metric systems.
* Responsive design.

### Built With

#### Technologies

* JavaScript
* CSS
* HTML
* WebPack
